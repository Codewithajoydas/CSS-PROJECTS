## Structure
Each project should have its own folder:
```
css-mastery-projects/
├── beginner/
│   └── 01-button-hover/
│       ├── index.html
│       └── style.css
├── intermediate/
│   └── 31-login-form/
│       ├── index.html
│       └── style.css
├── advanced/
│   └── 71-netflix-clone/
│       ├── index.html
│       └── style.css
└── README.md
```

## How to Use
- Clone the repo
- Navigate into any project folder
- Open `index.html` in a browser
- Modify and experiment with the `style.css`
